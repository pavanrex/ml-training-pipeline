# ML Training Pipeline

This project focuses on building an end-to-end machine learning pipeline, covering data preprocessing, training, evaluation, and basic optimization.

## What it does
- Loads and preprocesses dataset
- Trains a machine learning model
- Evaluates performance using standard metrics
- Tracks experiments and results

## Why I built this
I wanted to understand how machine learning systems work beyond just training a model — especially how data, training, and evaluation connect in a pipeline.
